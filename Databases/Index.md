## Best Revision Route

1. Read [COMP1004 Database Exam Guide.md](<COMP1004 Database Exam Guide.md>) for exam structure and priorities.
2. Work through [SQL and ERD Comprehensive Guide.md](<SQL and ERD Comprehensive Guide.md>) for the two biggest practical skills.
3. Read [Condensed Database Notes.md](<Condensed Database Notes.md>) once without trying to memorise.
4. Revisit the examples in [Lecture Slides](<Lecture Slides>) for ERD mapping, normalisation, and SQL queries.
5. Practise writing `CREATE TABLE`, `INSERT`, `UPDATE`, `DELETE`, and `SELECT` statements by hand.
6. Practise converting small ER diagrams into relational schemas.
7. Practise normalising a messy table into 1NF, 2NF, and 3NF.

## Highest Priority Topics

- Relational database vocabulary: table, relation, row, tuple, column, attribute, key, DBMS.
- ERD to relation mapping: entities, attributes, primary keys, foreign keys, relationships.
- Normalisation: functional dependencies, 1NF, 2NF, 3NF, and why redundancy causes problems.
- SQL DDL: `CREATE TABLE`, data types, constraints, `ALTER TABLE`, `DROP TABLE`.
- SQL DML: `SELECT`, `WHERE`, aggregate functions, `GROUP BY`, `HAVING`, joins, subqueries, string functions.
