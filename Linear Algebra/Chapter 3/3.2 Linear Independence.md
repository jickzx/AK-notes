## 3.2 Linear Independence

### The Definition of Linear Independence

#### Definition

A set of vectors $\{\mathbf{v}_1,\mathbf{v}_2,\dots,\mathbf{v}_k\}$ is [[Linear Independence|linearly independent]] if the vector equation

$$
x_1\mathbf{v}_1 + x_2\mathbf{v}_2 + \cdots + x_k\mathbf{v}_k = 0
$$

has only the trivial solution

$$
x_1 = x_2 = \cdots = x_k = 0.
$$

The set $\{\mathbf{v}_1,\mathbf{v}_2,\dots,\mathbf{v}_k\}$ is `linearly dependent` otherwise.

In other words, $\{\mathbf{v}_1,\mathbf{v}_2,\dots,\mathbf{v}_k\}$ is linearly dependent if there exist numbers $x_1,x_2,\dots,x_k$, not all equal to zero, such that

$$
x_1\mathbf{v}_1 + x_2\mathbf{v}_2 + \cdots + x_k\mathbf{v}_k = 0.
$$

This is called a `linear dependence relation` or `equation of linear dependence`.

### Checking Linear Independence

A set of vectors $\{\mathbf{v}_1,\mathbf{v}_2,\dots,\mathbf{v}_k\}$ is linearly independent if and only if the vector equation

$$
x_1\mathbf{v}_1 + x_2\mathbf{v}_2 + \cdots + x_k\mathbf{v}_k = 0
$$

has only the trivial solution, if and only if the [[Matrix Equation|matrix equation]]

$$
A\mathbf{x} = 0
$$

has only the trivial solution, where $A$ is the matrix with columns $\mathbf{v}_1,\mathbf{v}_2,\dots,\mathbf{v}_k$:

$$
A = \begin{bmatrix} \mathbf{v}_1 & \mathbf{v}_2 & \cdots & \mathbf{v}_k \end{bmatrix}.
$$

This is true if and only if $A$ has a [[Pivot|pivot]] position in every column.

So solving the matrix equation $A\mathbf{x}=0$ will either verify that the columns are linearly independent, or produce a linear dependence relation by substituting nonzero values for the [[Free Variable|free variables]].

### Example: Checking Linear Dependence

Is the set

$$
\left\{
\begin{pmatrix}
1\\
1\\
1
\end{pmatrix},
\begin{pmatrix}
1\\
-1\\
2
\end{pmatrix},
\begin{pmatrix}
3\\
1\\
4
\end{pmatrix}
\right\}
$$

linearly independent?

#### Solution

Equivalently, we are asking whether the homogeneous vector equation

$$
x
\begin{pmatrix}
1\\
1\\
1
\end{pmatrix}
+
y
\begin{pmatrix}
1\\
-1\\
2
\end{pmatrix}
+
z
\begin{pmatrix}
3\\
1\\
4
\end{pmatrix}
=
\begin{pmatrix}
0\\
0\\
0
\end{pmatrix}
$$

has a nontrivial solution. We solve this by forming a matrix and row reducing:

$$
\left(
\begin{array}{ccc}
1 & 1 & 3 \\
1 & -1 & 1 \\
1 & 2 & 4
\end{array}
\right)
\xrightarrow{\text{row reduce}}
\left(
\begin{array}{ccc}
1 & 0 & 2 \\
0 & 1 & 1 \\
0 & 0 & 0
\end{array}
\right).
$$

This says

$$
x = -2z
\quad \text{and} \quad
y = -z.
$$

So there exist nontrivial solutions. For instance, taking $z = 1$ gives the equation of linear dependence

$$
-2
\begin{pmatrix}
1\\
1\\
1
\end{pmatrix}
-
\begin{pmatrix}
1\\
-1\\
2
\end{pmatrix}
+
\begin{pmatrix}
3\\
1\\
4
\end{pmatrix}
=
\begin{pmatrix}
0\\
0\\
0
\end{pmatrix}.
$$

Therefore, the set is linearly dependent.

### Facts About Linear Independence

A wide matrix, meaning a matrix with more columns than rows, has linearly dependent columns.

For example, four vectors in $\mathbb{R}^3$ are automatically linearly dependent. A tall matrix may or may not have linearly independent columns.

More facts:

1. Two vectors are linearly dependent if and only if they are collinear, i.e. one is a scalar multiple of the other.
2. Any set containing the zero vector is linearly dependent.

### Criteria for Linear Independence

In this subsection we give two criteria for a set of vectors to be linearly independent. Keep in mind, however, that the actual definition is above.

#### Theorem

A set of vectors $\{\mathbf{v}_1,\mathbf{v}_2,\dots,\mathbf{v}_k\}$ is linearly dependent if and only if one of the vectors is in the [[Span|span]] of the other ones.

Any such vector may be removed without affecting the [[Span|span]].

#### Warning

In a linearly dependent set $\{\mathbf{v}_1,\mathbf{v}_2,\dots,\mathbf{v}_k\}$, it is not generally true that any vector $\mathbf{v}_j$ is in the span of the others, only that at least one of them is.

#### Theorem (Increasing Span Criterion)

A set of vectors $\{\mathbf{v}_1,\mathbf{v}_2,\dots,\mathbf{v}_k\}$ is linearly independent if and only if, for every $j$, the vector $\mathbf{v}_j$ is not in $\operatorname{Span}\{\mathbf{v}_1,\mathbf{v}_2,\dots,\mathbf{v}_{j-1}\}$.

We can rephrase this as follows:

If you make a set of vectors by adding one vector at a time, and if the span gets bigger every time you added a vector, then your set is linearly independent.
