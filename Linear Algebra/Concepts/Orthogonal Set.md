# Orthogonal Set

## Definition

A set of nonzero vectors is an [[Orthogonal Set|orthogonal set]] if every pair of distinct vectors in the set is orthogonal.

## Why It Matters

Orthogonal sets are automatically [[Linear Independence|linearly independent]], so they are excellent candidates for bases.

## Example

The set

$$
\left\{
\begin{pmatrix}
1\\
0
\end{pmatrix},
\begin{pmatrix}
0\\
1
\end{pmatrix}
\right\}
$$

is orthogonal because the dot product of the two vectors is $0$.
