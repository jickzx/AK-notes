# Matrix Transformation

## Definition

If $A$ is an $m\times n$ matrix, then

$$
T(\mathbf{x})=A\mathbf{x}
$$

defines a [[Matrix Transformation|matrix transformation]]

$$
T:\mathbb{R}^n\to\mathbb{R}^m.
$$

## Why It Matters

A matrix transformation lets us interpret a matrix geometrically as a function from one vector space to another.

Its [[Range|range]] is the [[Column Space|column space]] of the matrix.
