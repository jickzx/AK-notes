# Inverse Matrix

## Definition

A square matrix $A$ is [[Inverse Matrix|invertible]] if there exists a matrix $A^{-1}$ such that

$$
A^{-1}A=I
\qquad\text{and}\qquad
AA^{-1}=I.
$$

## Why It Matters

If $A$ is invertible, then the matrix equation

$$
A\mathbf{x}=\mathbf{b}
$$

has the unique solution

$$
\mathbf{x}=A^{-1}\mathbf{b}.
$$

So an inverse lets us solve linear systems directly and convert between coordinate systems.
