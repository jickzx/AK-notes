# Linear Independence

## Definition

A set of vectors $\{\mathbf{v}_1,\mathbf{v}_2,\dots,\mathbf{v}_k\}$ is linearly independent if the equation

$$
x_1\mathbf{v}_1 + x_2\mathbf{v}_2 + \cdots + x_k\mathbf{v}_k = 0
$$

has only the trivial solution.

The set is linearly dependent otherwise.

## Why It Matters

Linear independence tells us whether a set of vectors contains redundancy. If one vector can be built from the others, the set is dependent.
