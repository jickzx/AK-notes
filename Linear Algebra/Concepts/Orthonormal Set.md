# Orthonormal Set

## Definition

An [[Orthonormal Set|orthonormal set]] is an orthogonal set in which every vector has length $1$.

## Why It Matters

Orthonormal bases make coordinates especially easy to compute because coefficients come directly from dot products.

## Example

The standard basis vectors

$$
\mathbf{e}_1=
\begin{pmatrix}
1\\
0
\end{pmatrix},
\qquad
\mathbf{e}_2=
\begin{pmatrix}
0\\
1
\end{pmatrix}
$$

form an orthonormal set.
