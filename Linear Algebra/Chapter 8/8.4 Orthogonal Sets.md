## 8.4 Orthogonal Sets

### Orthogonal and Orthonormal Sets

A set of nonzero vectors is an [[Orthogonal Set|orthogonal set]] if every pair of distinct vectors is orthogonal.

It is an [[Orthonormal Set|orthonormal set]] if it is orthogonal and every vector has length $1$.

### Why They Are Useful

Orthogonal vectors are automatically [[Linear Independence|linearly independent]].

Also, coefficients in expansions become easy to compute.

If $\{\mathbf{u}_1,\dots,\mathbf{u}_m\}$ is an orthogonal basis, then

$$
\mathbf{x}=
\frac{\mathbf{x}\cdot \mathbf{u}_1}{\mathbf{u}_1\cdot \mathbf{u}_1}\mathbf{u}_1
+\cdots+
\frac{\mathbf{x}\cdot \mathbf{u}_m}{\mathbf{u}_m\cdot \mathbf{u}_m}\mathbf{u}_m.
$$

If the basis is orthonormal, this simplifies to

$$
\mathbf{x}=
(\mathbf{x}\cdot \mathbf{u}_1)\mathbf{u}_1+\cdots+(\mathbf{x}\cdot \mathbf{u}_m)\mathbf{u}_m.
$$

### Worked Example

Consider

$$
\mathbf{u}_1=
\begin{pmatrix}
1\\
0
\end{pmatrix},
\qquad
\mathbf{u}_2=
\begin{pmatrix}
0\\
1
\end{pmatrix}.
$$

Then

$$
\mathbf{u}_1\cdot \mathbf{u}_2=0,
$$

so the set is orthogonal. Since both vectors have length $1$, it is also orthonormal.

### Gram-Schmidt Process

Given a linearly independent set, the [[Gram-Schmidt Process|Gram-Schmidt process]] constructs an orthogonal basis with the same span.
