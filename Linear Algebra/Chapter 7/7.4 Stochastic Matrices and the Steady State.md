## 7.4 Stochastic Matrices and the Steady State

### Stochastic Matrices

A [[Stochastic Matrix|stochastic matrix]] is a matrix used to describe transitions between states in a probability model.

In the column-stochastic convention:

- each entry is nonnegative;
- each column sums to $1$.

### Steady State

A [[Steady State|steady-state vector]] $\mathbf{w}$ satisfies

$$
A\mathbf{w}=\mathbf{w}.
$$

So a steady state is an [[Eigenvector|eigenvector]] with [[Eigenvalue|eigenvalue]] $1$.

Usually we also require the entries of $\mathbf{w}$ to sum to $1$, so that it is a probability distribution.

### Worked Example

Suppose a person is each day in one of two states:

- state $1$: using app $X$
- state $2$: using app $Y$

Assume:

- if they are using app $X$ today, then tomorrow they stay with app $X$ with probability $0.75$ and switch to app $Y$ with probability $0.25$
- if they are using app $Y$ today, then tomorrow they switch to app $X$ with probability $0.25$ and stay with app $Y$ with probability $0.75$

Using the column-stochastic convention, the transition matrix is

$$
\text{first column = from app } {\color{orange} X},
\qquad
\text{second column = from app } {\color{cyan} Y}
$$

$$
A=
\begin{pmatrix}
{\color{orange} 0.75} & {\color{cyan} 0.25}\\
{\color{orange} 0.25} & {\color{cyan} 0.75}
\end{pmatrix}.
$$

To find a steady state, solve

$$
A\mathbf{w}=\mathbf{w}.
$$

This is equivalent to

$$
(A-I)\mathbf{w}=0.
$$

Writing $\mathbf{w}=\begin{pmatrix}x\\y\end{pmatrix}$ gives

$$
\begin{cases}
-0.25x+0.25y=0\\
0.25x-0.25y=0
\end{cases}
\quad \Rightarrow \quad x=y.
$$

Since probabilities must sum to $1$, we get

$$
x=y=\frac{1}{2}.
$$

So the steady state is

$$
\mathbf{w}=
\begin{pmatrix}
1/2\\
1/2
\end{pmatrix}.
$$

So in the long run, the probability distribution settles at:

- $50\%$ using app $X$
- $50\%$ using app $Y$

### Why This Matters

Repeated multiplication by a stochastic matrix often models the long-term distribution of a system. The steady state tells you where the system settles if the distribution stabilizes.
