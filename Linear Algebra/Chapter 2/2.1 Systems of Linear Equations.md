A system of linear equations is a collection of several linear equations. 

A [[Solution Set|solution set]] is the set which describes all the solutions of a system of equations. It is also the intersection of the graphical representations of the linear equations in $R^n$

![[Linear Algebra/_assets/two-equations-in-three-variables.png]]

### Parametric Description of Solution Sets

We can use parameters to describe equations. This is called an implicit equation. In this case, we used one parameter because the [[Solution Set|solution set]] is a line, but if the solution set were a plane, we might need to use two, so that you can describe any point on the plane by changing these two parameters. This idea is developed further in [[Linear Algebra/Chapter 2/2.3 Parametric Form]].
