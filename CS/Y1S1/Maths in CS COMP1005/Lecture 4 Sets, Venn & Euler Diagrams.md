# Set Theory

U is the set of all the things in the scope of the problem, the “universe”

A/B is the set of elements in A that are NOT in B

If A is {1,2,3} and U is {1,2,3,4,5}, U/A (like not A) is {4,5}

# Euler Diagrams

Euler Diagrams are Venn Diagrams without regions with no items in them

# Strict Subsets

> [!note] 🔥
> A strict subset is a subset that does not have all of the elements of the original set. Eg, {1,2} ⊂ {1,2,3}

An example of a subset that isn't strict is:

Let A = {1, 2} and B = {1, 2}.

Here, A $\subseteq$ B because every element of A is in B, but A is **equal** to B, so it's not a strict subset. Therefore, A $\subseteq$ B holds, but A $\not\subset$ B.
