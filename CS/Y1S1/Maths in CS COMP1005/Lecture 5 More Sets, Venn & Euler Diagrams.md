Empty set is denoted as ∅.

# Cardinality

> [!note] 🔥
> Cardinality is the number of elements in a set

Cardinality is denoted by vertical bars around a set. 

Eg: | A | = 3 if A = {0,1,2}

## Singleton

> [!note] 🔥
> A singleton is a set with a cardinality of 1, meaning it has one element.

## Powersets

> [!note] 🔥
> A powerset is the set of all subsets of an expression. 

Powersets are denoted with Greek rho. ∅ is always a member of powersets.

Eg: ρ(B) is the set of all subsets of B, or { X | X ⊆ B }.

The powerset of {a, b} is {∅,{a}, {b}, {a, b}}

cardinality of $|\rho(A)|$ = $2^{|A|}$

## Partitions

> [!note] 🔥
> A partition of a set is one that contains subsets of the original that share no elements with each other. 

Disjoint sets are sets with no common elements. 

We can split a set up into other disjoint non-empty sets, meaning each will have a subset of the old set. 

A partition of set S is a set containing n sets S1, S2, … Sn, such that:

Eg,  S = {1,2,3,4,5}

Partition examples: { {1,3,5}, {2,4} } or { {1},{2},{3},{4},{5} } (and others)
